Quick Tutorial

Open Builder Gallery

Get productive fast

Add components, place the spawn arrow, trace rays and export your schematic. This short guide covers the essentials.

Open Builder See Steps

1. Add components

Use the left sidebar in the Builder to add lenses, mirrors, planes and other items. Left click any component button to add it to the canvas. New items spawn at the arrow position.

2. Position & navigate

Left-click: Select component. Right-click and drag: Pan the canvas. Mouse wheel: Zoom in/out for precise placement. The arrow indicates where the next component will be placed - adjust its position as needed.

3. Trace rays

Toggle trace lines and rays from the top toolbar. Change ray shape, color and overlay multiple rays in the ray menu to compare different light paths through your optical system.

4. Edit & organize

Select an element to flip horizontally/vertically, hide or delete it. Use component actions in the toolbar: Flip X, Flip Y, Hide Selected, Show Selected, or Show All Components to restore hidden items.

5. Export & save

Export as SVG for figures or JSON to save/load complete schematics. Use the example gallery on the main page to import pre-built optical setups and learn from them.

Detailed Controls

Mouse Controls
  • Left click: Select component or add from sidebar
  • Right click + drag: Pan the canvas view
  • Mouse wheel: Zoom in/out
  • Click arrow: Reposition spawn point for new components
Component Actions
  • Delete Selected: Remove chosen component
  • Flip X/Y: Mirror component orientation
  • Hide/Show: Toggle component visibility
  • Show All: Reveal all hidden components
Render Options
  • Hide Trace Line: Toggle ray path visibility
  • Hide Rays: Toggle ray beam display
  • Both Rays: Show multiple ray types
  • Hide Grid: Toggle alignment grid
Export/Import
  • Export SVG: Save as vector image
  • Export JSON: Save complete schematic
  • Import JSON: Load saved schematic
  • Gallery: Import example layouts

Pro tips

  • Use Plane components and Hide Component feature to draw focus points between optical elements.
  • Flip components horizontally/vertically to quickly test different orientations.
  • Hide background elements when exporting images for clean presentations.
  • Position the spawn arrow strategically before adding components to maintain proper spacing.
  • Use the grid for precise alignment, then hide it for clean exports.
