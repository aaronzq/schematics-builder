How to Use

  1. Select a component from the left sidebar to add it to the canvas.
  2. Arrow indicates the position to spawn next component. Adjust arrow if needed.
  3. You can drag, rotate and change ray shape for each component. Just click on the component to reveal interaction handles.
  4. Use the buttons above the canvas to delete, hide, or show components and rays.
  5. Refresh webpage to start over.
  6. Tips: use Plane and Hide Component to achieve 4F system layout.
