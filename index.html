How to Use (on PC browser)

  1. Click a component from the left sidebar to add it to the canvas.
  2. Arrow indicates the position to spawn next component. Adjust arrow if needed.
  3. Left mouse: Select component. Right mouse: Pan the canvas. Wheel: Zoom in/out.
  4. Change ray shape, color and overlay multiple rays in the ray menu.
  5. Tips: use Plane and Hide Component to draw focus point between components
